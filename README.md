# CrewAI Multi-Agent Project

## Overview

This project uses CrewAI to build a multi-agent system where different AI agents collaborate to complete tasks. Each agent has a specific role and works together to achieve a common goal.

## Technologies Used

* Python
* CrewAI
* LangChain
* LLM (Large Language Model)

## Project Workflow

1. Define AI agents with specific roles
2. Assign tasks to each agent
3. Agents collaborate to complete the tasks
4. The final result is generated after all agents finish their work

## Example Agents

* Research Agent – Collects information
* Writer Agent – Generates content
* Reviewer Agent – Reviews and improves the output

## How to Run

Install required libraries:
pip install crewai

Run the project:
python main.py

## Result

The system demonstrates how multiple AI agents can work together to solve complex tasks efficiently.
